# 📅 TO-DO / Appointment Management Application

## 📌 Project Overview

This project is a **TO-DO / Appointment Management Application** that allows users to manage their daily tasks or appointments efficiently.

It includes **user authentication** and a **dashboard system** where users can perform full **CRUD operations (Create, Read, Update, Delete)** on appointments.

---

## 🚀 Features

### 🔐 User Authentication
- User Registration (Sign Up)
- User Login (Sign In)
- Secure dashboard access

---

### 📊 Dashboard
- Displays all user appointments
- Easy navigation and management

---

### 📝 Appointment Management

Users can perform the following actions:

- 👁️ View Appointments  
- 🔍 Filter Appointments  
- 📊 Sort Appointments  
- ➕ Add Appointment  
- ✏️ Edit Appointment  
- ❌ Delete Appointment  

---

## 🛠️ Tech Stack

- Frontend: React.js  
- Backend: Javascript  
- Database: JSON Server   
- Styling: CSS / Bootstrap
